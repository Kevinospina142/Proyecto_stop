1) El sistema le mostrara las reglas del juego
2) El sistema le mostrara un texto donde le preguntara al jugador si desea jugar o no, el usuario debera ingresar una s si desea jugar o n si no desa jugar
3) El sistema le preguntara cuantas rondas desea jugar con un maximo de 10 rondas, el usuario ingresara un numero el cual sera las rondas
4) El sistema le pedira al usuario que digite su nombre
5) El sistema le dara la bienvenida y le dira con cual letra debe jugar la primera ronda
6) El sistema le mostrara el turno en el que va y le pedira el nombre con esa letra
7) El sistema le pedira un apellido con esa letra
8) El sistema le pedira una ciudad con esa letra
9) El sistema le pedira un color con esa letra
10) El sistema le pedira una cosa con esa letra
11) el sistema va a volver a mostrar el turno y volver a pedir los datos pero con una letra diferente. Esto sucedera hasta que se haya completado los turnos ingresados anteriormente
12) al finalizar el sistema mostrara una tabla con el nombre del jugador, los datos ingresados por el jugador, los aciertos y desaciertos que obtuvo, los puntos en total que consiguio y el tiempo en el que se demoro
13) el sistema mostrara las estadisticas totales, las cuales serian: el tiempo total de la ejecucion del programa, el tiempo total consumido por el jugador, los puntos totales,
los fallos totales y los aciertos totales
